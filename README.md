# CalculatorProject
# Simple Calculator using Python Tkinter 🧮

A basic GUI Calculator built using Python and Tkinter. This calculator performs simple arithmetic operations such as addition, subtraction, multiplication, and division.

## Features

* User-friendly graphical interface
* Addition (+)
* Subtraction (-)
* Multiplication (*)
* Division (/)
* Clear button (C)
* Error handling for invalid expressions
* Instant result calculation

## Technologies Used

* Python 3
* Tkinter (Python GUI Library)

## How to Run

1. Clone the repository:

```bash
git clone <your-repository-url>
```

2. Navigate to the project folder:

```bash
cd CalculatorProject
```

3. Run the application:

```bash
python calculator.py
```

## Sample Interface

```text
 -----------------
|               |
|      Display  |
 -----------------
| 7 | 8 | 9 | / |
| 4 | 5 | 6 | * |
| 1 | 2 | 3 | - |
| 0 | . | C | + |
 -----------------
|       =        |
 -----------------
```

## How It Works

* Click the number and operator buttons to form an expression.
* Press **=** to evaluate the expression.
* Press **C** to clear the display.
* If an invalid expression is entered, the calculator displays **Error**.
